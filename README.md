# HtMetadataDecrypt
simple metadata repair tool i made while working on my crackproof unpacker
games this has been tested on:
* Umamusume: Pretty Derby (Steam, global)

if this does not work on a game and they don't do anything else special (i think uma:pd jp uses something special), [make an issue](https://github.com/notunixian/HtMetadataDecrypt/issues?q=sort%3Aupdated-desc+is%3Aissue+is%3Aopen)

usage:
`HtMetadataDecrypt.exe <input_file> <output_file>`
